## ♟️ Features

Here's what you can do in ChessGame:

- **Chess Board UI:** Interactive 8x8 chess board with a clean layout
- **Piece Movement:** Basic chess piece movement logic implemented
- **Turn-Based Gameplay:** Players take turns (White vs Black)
- **Piece Selection:** Click to select and move pieces
- **Game State Handling:** Tracks moves and updates board dynamically

---

## ⚙️ The Process

I started by creating the chess board UI using WPF and XAML to build a structured grid layout.

Next, I implemented the core game logic, including piece movement and turn handling.  
This required defining how each chess piece behaves and validating legal moves.

After that, I focused on user interaction — selecting pieces, highlighting moves, and updating the board dynamically.

To keep the code clean and maintainable, I structured the application using patterns like MVVM and separated logic from UI.

Throughout the project, I focused on building a simple but scalable foundation that can be extended with more advanced chess features.
